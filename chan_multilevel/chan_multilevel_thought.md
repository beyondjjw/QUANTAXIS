# 缠论多级别联立分析 - 核心思想

## 1. 基本原理

```
多级别联立 = 不同周期K线的缠论分析 + 综合判断
```

**核心**: 通过大周期定方向，小周期找买卖点

---

## 2. 级别体系

| 级别 | 周期 | 作用 |
|------|------|------|
| 年线 | 12mon | 长期趋势判断 |
| 季线 | 季线 | 中期判断 |
| 月线 | 1mon | 方向判断 |
| 周线 | 1week | 波段操作 |
| **日线** | **D** | **主操作周期** |
| 30分钟 | 30min | 精确入场 |
| 5分钟 | 5min | 细节处理 |
| 1分钟 | 1min | 极值点位 |

---

## 3. 联立逻辑

```
┌─────────────────────────────────────┐
│           大级别 (日线+)             │
│  判断方向: 上涨/下跌/盘整            │
│  判断背驰: 是否出现                  │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│           小级别 (30分/5分)          │
│  寻找精确买卖点                      │
│  确认大级别方向                      │
└──────────────┬──────────────────────┘
               │
               ▼
           综合信号
```

---

## 4. 买卖点体系

### 买点

| 买点 | 条件 | 级别 |
|------|------|------|
| **一买** | 大级别下跌背驰终点 | 日线/周线 |
| **二买** | 大级别上涨中，小级别下跌不破前低 | 30分 |
| **三买** | 大级别上涨回调不破中枢上沿 | 30分/5分 |

### 卖点

| 卖点 | 条件 |
|------|------|
| **一卖** | 大级别上涨背驰 |
| **二卖** | 大级别下跌反弹高点 |
| **三卖** | 大级别下跌反弹破中枢下沿 |

---

## 5. 背驰检测

```
背驰 = 创新高/新低 + 力度减弱

表现:
- 股票: 黄白线创新高但面积减少
- 期货: 柱子面积减少
```

---

## 6. 策略框架

```python
# 伪代码
def multi_level_signal(analyzer):
    big = analyzer.get_big_level()   # 日线
    small = analyzer.get_small_level() # 30分
    
    # 大级别上涨 + 小级别背驰 → 二买/三买
    if big.direction == '上涨' and small.beichi:
        return 'buy'
    
    # 大级别下跌 → 观望
    if big.direction == '下跌':
        return 'watch'
    
    # 大级别背驰 → 可能见顶
    if big.beichi:
        return 'sell'
```

---

## 7. 核心要点

| 要点 | 说明 |
|------|------|
| **先大后小** | 先看大周期方向，再找小周期机会 |
| **多周期验证** | 不同周期相互印证 |
| **背驰确认** | 背驰是转折的关键信号 |
| **中枢为锚** | 中枢是分析的核心参考 |
| **量价配合** | 结合成交量判断有效性 |

---

## 8. 应用场景

1. **选股**: 日线下跌背驰 → 30分出现二买
2. **入场**: 30分回调结束点买入
3. **持有**: 30分上涨延续中持有
4. **卖出**: 30分背驰 或 日线背驰
