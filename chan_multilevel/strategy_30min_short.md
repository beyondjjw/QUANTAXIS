# 30分钟级别短线波段策略

## 一、核心理念

> **30分钟主战 + 5分钟精准 + 15分钟辅助**
> - 做30分钟级别的笔结构完整行情
> - 不做日内频繁交易
> - 波段操作，快进快出

---

## 二、级别配置

| 级别 | 周期 | 作用 |
|------|------|------|
| 30分钟 | **主操作周期** | 笔/中枢/背驰/买卖点 |
| 15分钟 | 辅助判断 | 方向验证 |
| 5分钟 | 精确入场 | 1分钟笔结构 |

---

## 三、方向判断

### 3.1 30分钟方向

```python
def get_30min_direction(bars):
    # 使用EMA判断
    ema12 = bars['close'].ewm(span=12).mean()
    ema26 = bars['close'].ewm(span=26).mean()
    
    if ema12.iloc[-1] > ema26.iloc[-1]:
        return '上涨'
    return '下跌'
```

### 3.2 趋势判断

```python
def get_30min_trend(bars):
    ma20 = bars['close'].rolling(20).mean().iloc[-1]
    ma60 = bars['close'].rolling(60).mean().iloc[-1]
    
    if ma20 > ma60:
        return '上涨趋势'
    elif ma20 < ma60:
        return '下跌趋势'
    return '盘整'
```

## 四、买卖点体系

### 4.1 买点 (30分钟)

| 买点 | 条件 | 仓位 |
|------|------|------|
| **底分型买** | 30分底分型确认 | 30% |
| **二买** | 30分上涨中回调不破前低 | 50% |
| **三买** | 突破30分中枢后回调不破 | 20% |

### 4.2 卖点 (30分钟)

| 卖点 | 条件 |
|------|------|
| **顶分型卖** | 30分顶分型确认 |
| **背驰卖** | 30分上涨背驰 |
| **止损卖** | -3%止损 |

---

## 五、风控规则

### 5.1 仓位控制

| 情况 | 仓位 |
|------|------|
| 上涨趋势中 | 60-80% |
| 盘整中 | 30-50% |
| 下跌趋势中 | 0-20% |

### 5.2 止损规则

| 类型 | 幅度 |
|------|------|
| 买入后止损 | -3% |
| 盘整破坏止损 | -2% |
| 收盘破MA20 | -2% |

### 5.3 止盈规则

| 条件 | 卖出 |
|------|------|
| 涨幅>8% | 50% |
| 出现顶分型 | 30% |
| 背驰 | 20% |

## 六、实战流程

```
1. 15分钟方向确认
   └─ 上涨? → 继续

2. 30分钟笔结构
   └─ 有完整笔? → 等待回调

3. 5分钟精准买入
   └─ 1分钟底分型 → 买入

4. 持仓监控
   └─ 30分顶分型/背驰 → 卖出

5. 止损/止盈
   └─ -3%止损 / +8%止盈
```

---

## 七、关键要点

1. **只做30分钟笔完整行情** - 不追瞬间涨跌
2. **不隔夜** - 收盘前处理仓位
3. **3%止损** - 短线必须严格止损
4. **8%止盈** - 达到目标分批卖
5. **日交易不超过3次** - 避免过度交易
