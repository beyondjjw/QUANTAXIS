         │

## 6. 核心类说明

### 数据对象 (py/objects.py)

| 类名 | 说明 | 主要属性 |
|-----|------|---------|
| `RawBar` | 原始K线 | symbol, dt, open, close, high, low, vol |
| `NewBar` | 处理后K线 | 去除包含关系 |
| `FX` | 分型 | dt, Mark (顶/底), high, low |
| `BI` | 笔 | start_dt, end_dt, direction (向上/向下) |
| `ZS` | 中枢 | start_dt, end_dt, high, low |
| `Signal` | 交易信号 | name, value, direction |

### 分析器 (py/analyze.py)

| 类名 | 说明 |
|-----|------|
| `CZSC` | 缠论分析器 - 核心类 |
| `BarGenerator` | K线生成器 - 多周期转换 |

### 信号 (signals/)

| 信号 | 说明 | 文件 |
|-----|------|------|
| `CS` | 缠论信号基类 | signals.py |
| `FS` | 快速信号 | signals.py |
| `bxt` | 笔系统信号 | bxt.py |
| `ta` | 技术指标信号 | ta.py |
| `cat` | CAT 信号 | cat.py (你的定制) |
| `vol` | 量价信号 | vol.py |
| `pos` | 仓位信号 | pos.py |

### 交易 (traders/)

| 类名 | 说明 |
|-----|------|
| `CzscTrader` | 实盘交易员 |
| `CzscSignals` | 信号交易员 |
| `WeightBacktest` | 权重回测 |

### AI 选股 (ai/)

| 类名 | 说明 |
|-----|------|
| `AISelector` | 机器学习选股器 |
| `ChanSignalAI` | 缠论+AI 信号 |
| `EnsembleSelector` | 集成选股器 |
