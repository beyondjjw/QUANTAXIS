# QUANTAXIS 项目概述

![项目全景](./images/02-project-overview.png)

## 项目目标

QUANTAXIS 是一个开源的量化金融框架（Quantitative Financial Framework），旨在为量化交易者提供完整的数据获取、策略回测、实盘交易的一站式解决方案。项目 Fork 自 yutiansut/QUANTAXIS，致力于构建专业、易用的量化投资工具。

## 核心用户

- **个人量化投资者** — 自主开发策略并进行回测和实盘交易
- **量化研究团队** — 进行因子研究、策略开发与组合分析
- **金融机构** — 构建量化交易系统和风控平台

## 系统边界

QUANTAXIS 与以下外部服务/系统交互：

- **MongoDB** — 主要数据存储（行情数据、策略结果、账户数据）
- **ClickHouse** — 时序数据分析与 OLAP 查询
- **RabbitMQ** — 消息队列，用于事件驱动架构
- **Tushare** — A股市场数据源
- **Tdx (pytdx)** — 行情数据接口（通达信）
- **交易所 API** — 实盘交易接口（期货、股票、数字货币）

## 核心业务流程

1. **数据获取与存储流程**
   - 通过 QAFetch 模块从多数据源（TuShare、Tdx、交易所 API）获取行情数据
   - 数据经清洗后存储到 MongoDB/ClickHouse
   - 支持增量更新和全量刷新

2. **策略回测流程**
   - 用户编写策略（继承 QAStrategy 基类）
   - QAEngine 加载历史数据进行回测
   - 生成绩效分析报告（收益、回撤、夏普比率等）

3. **实盘交易流程**
   - 通过 QIFI 统一账户体系连接多市场
   - 策略信号通过 QAMarket 发单
   - 实时风控和仓位管理
